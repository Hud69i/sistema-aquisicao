\# 📚 Sistema de Aquisição - Livraria



Este é um projeto acadêmico desenvolvido em \*\*Linguagem C\*\* para gerenciamento e cadastro de livros. O programa permite a entrada de dados e gera um relatório formatado para o utilizador.



\## 🚀 Funcionalidades

\* Cadastro de Título do Livro.

\* Cadastro de Autor da Obra.

\* Cadastro de Número de Páginas.

\* \*\*Validação de Entrada:\*\* O sistema verifica se o valor de páginas é numérico para evitar erros de execução.

\* Exibição de um relatório organizado em tabela.



\## 🛠️ Tecnologias Utilizadas

\* \*\*Linguagem C / C++\*\*

\* \*\*IDE:\*\* Dev-C++ 5.11

\* \*\*Compilador:\*\* TDM-GCC 4.9.2



\## 📖 Como utilizar

1\. Abra o arquivo `sistema-aquisicao.dev` no Dev-C++.

2\. Pressione `F11` para compilar e executar.

3\. Siga as instruções no terminal para cadastrar o livro.
